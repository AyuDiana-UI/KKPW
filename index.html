Cerita Selama Semester Ganjil

Saya mengalami banyak pengalaman baru saat di semester satu karena di semester ini saya resmi naik kelas 11. Di semester ganjil ini saya menyadari bahwa beban saya di jurusan ini akan bertambah. Awalnya saya cukup bingung dan takut tidak bisa mengikuti pembelajaran di semester ganjil dengan baik karena dihadapkan dengan materi baru. Tapi lama kelamaan saya mulai terbiasa dan beradaptasi dengan lingkungan yang baru. Di semester ini juga saya mendapat pengalaman baru saat ulang tahun sekolah. Saat itu ekstra saya diminta untuk mengisi acara hiburan saat ulang tahun sekolah. Di kegiatan ini juga saya beberapa kali mengambil dispen untuk mempersiapkan penampilan. Kadang latihan bisa menghabiskan waktu hingga jam 6 sore. Mendekati akhir tahun, berarti sudah mendekati sumatif akhir semester. Sebelum sumatif, kami menghadapi kokurikuler terlebih dahulu. Saya dan kelompok saya bekerja sama mengerjakan kokurikuler yang sebetulnya sangat banyak tapi harus diselesaikan dalam waktu 3 hari. Kami berhasil menyelesaikannya dengan baik dan tepat waktu. Minggu depannya, kami mendapat libur hari raya selama 2 minggu. Setelah libur selesai, kami langsung mengerjakan sumatif. Cukup sulit bagi saya karena saya ditugaskan mengerjakan beberapa project. Ditambah lagi saya remedial di salah satu mata pelajaran. Cukup fatal bagi saya karena itu merupakan mata pelajaran kejuruan. Saya dan kelompok saya berhasil mengerjakan sebuah project game yang ditugaskan meski terhalang banyak kendala.

Di saat-saat ini cukup sangat sulit bagi saya. Terutama saat sumatif akhir yang diadakan setelah libur hari raya. Jadinya waktu saya dirumah terbagi menjadi dua. Untuk mempersiapkan hari raya dan belajar karena rumah saya memerlukan banyak banten saat hari raya. Jadi saya tidak tenang karena teringat terus dengan sumatif akhir. Saran saya mungkin ke depannya sumatif akhir bisa diadakan sebelum hari raya agar bisa tenang saat melaksanakan hari raya.

Cerita Selama Liburan 2 Minggu

Kegiatan saya di rumah selama libur tidak banyak. Saya bangun lebih siang tapi masih tahu waktu karena saya harus membersihkan area halaman rumah dan juga tempat suci. Saya juga membantu untuk menghaturkan banten. Setelah itu, saya membantu ibu untuk berjualan di toko hingga sore. Kadang saya juga menyempatkan diri untuk keluar bersama teman agar tidak bosan di rumah. Disaat menjaga toko, saya juga sering menonton film. Tidak hanya saat menjaga toko, tapi juga sebelum tidur untuk mengisi waktu luang saya. Tapi terkadang saya tidur larut malam karena terlalu lama bermain hp dan nonton film.

Di masa liburan ini, saya hanya fokus dalam pekerjaan rumah sehingga lupa untuk mengeksplor minat saya. Seharusnya saya bisa mempelajari jurusan keahlian saya agar tidak stuck disana-sana saja.

Harapan

Harapan saya di semester ini adalah saya bisa mengikuti pembelajaran dan mengerjakan semua project dengan baik dan tepat waktu. Saya juga ingin mencari minat dan keahlian yang benar benar sejalan dengan saya agar nanti saat kuliah, saya tidak salah mengambil jurusan.

Saya berharap ke depannya komunikasi antara siswa dengan siswa dan siswa dengan guru dapat semakin terbuka, sehingga proses belajar menjadi lebih nyaman dan menyenangkan.
